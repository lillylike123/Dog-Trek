# Dog Trek

This is my first-ever 2D game made with JavaScript, and I learned some features from Frank's Laboratory

I loved making this project; it's my first full-fledged project after learning JavaScript, HTML, and CSS over the past week. I am so glad I was able to move away from the bite-sized projects I used to do and move into a web-playable game made using my new favourite coding language. I think this week has definitely been a learning curve, and I can't wait to see what coding has in store for me in the future.

I added unique features to make this game special and set it apart from the rest.

Controls
-----------
Right Arrow - Forward
Up Arrow - Jump
Enter - Roll & Attack
Down Arrow - Smash (When in the sky)
Reload - Restart Game

Goal
-----------
In 30 seconds, you must kill 20 enemies and have a score of 20
