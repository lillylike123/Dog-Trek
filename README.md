# Dog Trek

This is my first-ever 2D game made with JavaScript, and I initially followed a tutorial from FreeCodeCamp made by Frank's Laboratory. 

(https://www.youtube.com/watch?v=GFO_txvwK_c)

Although I started by following this tutorial, I added my own features and parts:

- Sound effects
- Levels
- Game gets harder the more you play
- Camera shake when hit
- Gameplay background 
- Different winning conditions

This makes my project unique and special, and isn't just any generic game that follows this specific tutorial/


I loved making this project; it's my first full-fledged project after learning JavaScript, HTML, and CSS over the past week. I am so glad I was able to move away from the bite-sized projects I used to do and move into a web-playable game made using my new favourite coding language. I think this week has definitely been a learning curve, and I can't wait to see what coding has in store for me in the future.

I added unique features to make this game special and set it apart from the rest.

Controls
-----------

Right Arrow - Forward/Go faster

Up Arrow - Jump

Enter - Roll & Attack

Down Arrow - Smash (When in the sky)

Reload - Restart Game

Goal
-----------

In 30 seconds, you must kill 20 enemies and have a score of 20

Thanks for checking this out <3
=======

You must get 50 points in 90 seconds to win the game.
